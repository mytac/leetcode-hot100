11. 盛最多水的容器

[连接](https://leetcode-cn.com/problems/container-with-most-water/)

给你 n 个非负整数 a1，a2，...，an，每个数代表坐标中的一个点  (i, ai) 。在坐标内画 n 条垂直线，垂直线 i  的两个端点分别为  (i, ai) 和 (i, 0) 。找出其中的两条线，使得它们与  x  轴共同构成的容器可以容纳最多的水。

说明：你不能倾斜容器。

## eg1

```
输入：[1,8,6,2,5,4,8,3,7]
输出：49
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。
```

提示

```
n = height.length
2 <= n <= 3 * 104
0 <= height[i] <= 3 * 104
```

## 分析

### 法一：暴力解法-超出时间限制，但能解出来

```c++
int maxArea(vector<int>& height) {
  int i=0,len=height.size(),j;
  int temp=0,max=0;
  for(i=0;i<len;i++){
    for(j=i;j<len;j++){
      temp=MIN(height[i],height[j])*(j-i);
      if(temp>max){
        max=temp;
      }
    }
  }
  return max;
}
```

## 法二：双指针

复杂度为 O(n)，和上面的思路一样，求 2 指针指向元素的最小值 乘以 两指针之间的距离。只有移动指针的方法是不一样的，移动指针时，只移动元素小的那个指针。

把整个问题想成几何问题，初态时底是最大的，之后的过程不管指针怎么移动，底都是递减的，所以要使面积更大，只能保留高是最大值的那根指针，因此使值小的指针进行移动。在这个过程中可解。
